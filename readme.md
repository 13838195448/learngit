分布式版本控制系统  git  BitKeeper Mercurial  Bazaar


		创建版本库  mkdir 文件夹名称   
		进入文件夹   cd 文件夹名称
		初始化git    git init
		
	把文件添加到仓库  git add  文件名	
	把文件提交到仓库  git commit -m "本次提交的说明"
												例如：	$ git add file1.txt
														$ git add file2.txt file3.txt
														$ git commit -m "add 3 files."	
								
	
	git status 查看仓库当前的状态	
	git diff 文件名  查看不同
